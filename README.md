# Market Fetcher API (PHP)

A lightweight PHP-based API for fetching real-time gold, coin, and fiat currency prices from [tgju.org](https://www.tgju.org).

## Quick Start

```bash
php -S localhost:8000 -t public
```

Then open in your browser:

* [http://localhost:8000/gold](http://localhost:8000/gold)
* [http://localhost:8000/coin](http://localhost:8000/coin)
* [http://localhost:8000/currency](http://localhost:8000/currency)
* [http://localhost:8000/all](http://localhost:8000/all)

Or using query params:

* [http://localhost:8000/?type=gold](http://localhost:8000/?type=gold)
